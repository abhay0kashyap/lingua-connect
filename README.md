## 🚀 Overview

**Lingua Connect** is an AI-powered application that enables intelligent language interaction using **Google Gemini**.  
It allows users to experiment with conversational AI, language understanding, and generative responses in a clean and simple interface.

This project is designed for:
- Learning modern AI integration
- Building real-world AI-powered applications
- Experimenting with Gemini models via AI Studio

---

## ✨ Features

- 🤖 AI-powered conversations using **Gemini API**
- ⚡ Fast development setup with modern tooling
- 🧠 Clean and minimal UI for interaction
- 🔑 Secure API key handling via environment variables
- 🌍 Easily extendable for translation, chatbots, or assistants

---

## 🛠️ Tech Stack

- **Frontend / App Framework:** Node.js
- **AI Model:** Google Gemini
- **API Platform:** Google AI Studio
- **Environment Management:** `.env.local`

---

## 📦 Prerequisites

Before running the project, make sure you have:

- **Node.js** (v18 or later recommended)
- **npm** or **yarn**
- A **Google Gemini API Key**

---

#

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
